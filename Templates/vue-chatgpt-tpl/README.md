# vue-chatgpt-tpl

## Bio

A ChatGPT template based on OpenAI API (gpt-3.5-turbo).

## Configuration Guide

Input the following parameters when configuring the template

- PageTitle
- Bio
- OpenAI API Key
- Logo
- Avatar
- Button Link
